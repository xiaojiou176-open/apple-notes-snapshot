# Support

Use the support lane that matches your situation:

- **Start here / orientation**
  Use the General discussion:
  <https://xiaojiou176-open.github.io/apple-notes-snapshot/support/>
- **Setup help or troubleshooting**
  Use the Q&A discussion:
  <https://github.com/xiaojiou176-open/apple-notes-snapshot/discussions/categories/q-a>
- **Feature ideas**
  Use the Ideas discussion:
  <https://github.com/xiaojiou176-open/apple-notes-snapshot/discussions/categories/ideas>
- **Show your workflow**
  Use the Show and tell discussion:
  <https://github.com/xiaojiou176-open/apple-notes-snapshot/discussions/categories/show-and-tell>
- **Sensitive security issues**
  Use private vulnerability reporting:
  <https://github.com/xiaojiou176-open/apple-notes-snapshot/security/advisories/new>

Open a GitHub issue when you have a reproducible bug in the tracked repository
surface and can provide clear reproduction steps.
