# Code of Conduct

## Our Pledge

We want Apple Notes Snapshot to be a respectful, calm, and useful open-source
project for maintainers, contributors, and security reporters.

Participants in this project are expected to:

- be respectful in public and private communication
- assume good intent while staying clear and direct
- focus feedback on the work, not on the person
- avoid harassment, intimidation, discrimination, or personal attacks
- respect maintainer time and the project scope

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, hate speech, or discriminatory language
- threats, stalking, or intimidation
- publishing private information without permission
- posting exploit details, secrets, personal note content, or sensitive logs in
  public issues
- repeated bad-faith arguments or disruptive behavior after moderator requests
  to stop

## Reporting

For behavior concerns, open a minimal private contact request through the
repository issue templates if a safer path is needed, or contact the repository
admins through GitHub-maintained moderation/reporting flows.

For security issues, use the flow documented in `SECURITY.md` instead of public
discussion.

## Enforcement

Maintainers may remove comments, close discussions, or block participation when
behavior violates this code of conduct or creates risk for contributors,
maintainers, or security reporters.
