Latest release

v0.1.7 turns starter packs and public skills into tagged truth.

This release keeps Apple Notes Snapshot in its control-room-first lane while packaging the host-facing guidance into clearer, more reusable surfaces and tightening the launch-truth boundary.

Highlights

  • Added dedicated Codex and Claude Code starter-pack pages.
  • Added a tracked public-skills pack with a curated public-safe subset.
  • Split GitHub-controlled release and announcement actions from truly external-only launch work.

Why it matters

The host ecosystem now feels more productized without pretending the repo ships official plugins, a public SDK, or attach-verified host support. The release and launch story also got cleaner: GitHub actions already taken are now clearly separated from Search Console, DNS, video, and social actions that still require owner-side control.
