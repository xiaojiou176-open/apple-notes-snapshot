# Apple Notes Snapshot v0.1.10

## What changed after v0.1.9

This release turns the second-pass curation polish into tagged truth.

## Highlights

- deepened the curated storytelling across Home, Proof, For Agents, Releases, and Quickstart
- strengthened the visual language for proof ledgers, release storytelling, and builder-lane sequencing
- kept the control-room-first product identity fully intact while making the docs feel more intentionally exhibited

## Why it matters

This release does not change what Apple Notes Snapshot is. It changes how
quickly a new reader can feel that it is coherent, trustworthy, and worth
serious attention. In plain language: the repo now reads more like a curated
exhibit and less like a stack of individually good pages.
