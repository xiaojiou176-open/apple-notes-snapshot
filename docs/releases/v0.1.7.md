# Apple Notes Snapshot v0.1.7

## What changed after v0.1.6

This release turns the new builder-facing packaging work into tagged truth.

## Highlights

- added dedicated Codex and Claude Code starter-pack pages with install, attach,
  verify, and proof-boundary guidance
- added a tracked public-skills pack with a curated public-safe subset instead
  of exposing the repo-local `.agents/skills/` tree
- split GitHub-controlled release and announcement actions from truly
  external-only Search Console, domain, video, and social work

## Why it matters

This release makes the host ecosystem feel more productized without pretending
the repo ships official plugins, a public SDK, or attach-verified host support.
In plain language: the wiring kit now comes in labeled boxes, and the launch
checklist now clearly separates buttons we already pressed from the ones only an
owner can press.
