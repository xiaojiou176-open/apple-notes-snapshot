# Apple Notes Snapshot v0.1.9

## What changed after v0.1.8

This release closes the gap between “the repo is strong” and “the public surface
proves it cleanly.”

## Highlights

- added a public proof page for repo-side gates, GitHub-controlled evidence,
  live host-local surfaces, and trust boundaries
- tightened the front door so the control-room story stays first and integration
  surfaces stay second-lane
- hardened launchd install against disabled-label host state
- stopped lock-contention skip paths from polluting the state ledger as
  `aborted`

## Why it matters

This release does not try to make Apple Notes Snapshot a bigger product. It
makes it a cleaner one. In plain language: public truth is now synced, the
proof trail is easier to read, and the local scheduler path is more resilient
on real machines.
