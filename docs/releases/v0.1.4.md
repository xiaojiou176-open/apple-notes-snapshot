# Apple Notes Snapshot v0.1.4

## What changed after v0.1.3

This release makes the public story more future-aware and more internally
consistent.

## Highlights

- a roadmap page that says what is next without pretending the project is becoming a cloud sync product
- stale announcement and release references cleaned up across the docs surface
- a stronger "what next?" path from the landing page into roadmap and release history

## Why it matters

Good open-source presentation is not just about launch day. Visitors also want
to know:

- what is current
- what is likely next
- what is explicitly out of scope

`v0.1.4` makes those answers easier to find.
