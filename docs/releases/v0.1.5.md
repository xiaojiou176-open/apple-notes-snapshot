# Apple Notes Snapshot v0.1.5

## What changed after v0.1.4

This release improves the "first successful run" path for new visitors.

## Highlights

- a public examples page that shows believable local workflows
- a public troubleshooting page for permissions, launchd, stale-run, and Web UI issues
- stronger navigation from the landing page, roadmap, support, and community pages into those surfaces

## Why it matters

People do not only need to understand a project. They also need to picture
where it fits and what to do when the first run is not smooth. `v0.1.5`
reduces that gap.

## Current main branch note

This `v0.1.5` tag predates the current main-branch AI Diagnose, Local Web API,
MCP Provider, For Agents surfaces, and builder-friendly entry points for
Codex- and Claude Code-style local workflows.
