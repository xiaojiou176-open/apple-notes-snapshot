Second-pass curation polish

v0.1.10 makes the public docs feel more like a designed exhibit.

This release keeps the control-room-first product identity completely intact while making Home, Proof, For Agents, Release history, and Quickstart read with more visual rhythm, clearer sequencing, and stronger curation.

Highlights

  • Deepened the curated storytelling across Home, Proof, For Agents, Releases, and Quickstart.
  • Strengthened the visual language for proof ledgers, release storytelling, and builder-lane sequencing.
  • Kept the control-room-first product identity fully intact while making the docs feel more intentionally exhibited.

Why it matters

This release does not change what Apple Notes Snapshot is. It changes how quickly a new reader can feel that it is coherent, trustworthy, and worth serious attention. In plain language: the repo now reads more like a curated exhibit and less like a stack of individually good pages.
