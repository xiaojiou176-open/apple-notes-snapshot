Public truth, tightened

v0.1.9 turns the repo’s strongest traits into a cleaner public contract.

This release does not expand Apple Notes Snapshot into a bigger platform. It makes the existing control-room, proof, and scheduler story easier to trust: the public truth is synced, the proof trail is centralized, and disabled launchd labels on the current host no longer block install from self-recovering.

Highlights

  • Added a public proof page for repo-side gates, GitHub-controlled evidence, live same-machine surfaces, and trust boundaries.
  • Tightened the front door so the control-room story stays first and builder surfaces stay second-lane.
  • Hardened launchd install against disabled-label host state.
  • Stopped lock-contention skip paths from polluting the state ledger as aborted.

Why it matters

This release closes the gap between “the repo is strong” and “the public surface proves it cleanly.” In plain language: it is easier to see what is real, what is local-only, and what is already verified without reading the whole repo by hand.
