# Apple Notes Snapshot v0.1.2

## What changed after v0.1.1

This release makes the public project surface easier to navigate as the repo
keeps evolving.

## Highlights

- a dedicated community hub page that explains which discussion lane to use
- a release-history page that links the public site, GitHub releases, and the
  changelog together
- a changelog upgraded from a launch note into a real multi-version public
  history

## Why it matters

The first releases proved the project existed and was active. `v0.1.2` makes it
easier for a new visitor to understand the maintenance story at a glance:

- where to ask questions
- where to share ideas
- where to follow releases
- where to read the official version history
