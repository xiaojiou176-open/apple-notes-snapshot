History pass

v0.1.2 made the public maintenance story easier to follow.

This release tied the docs site, release history, and changelog together so a new visitor could understand the maintenance trail at a glance.

Highlights

  • A dedicated community hub page that explains which discussion lane to use.
  • A release-history page that links the public site, GitHub releases, and the changelog together.
  • A changelog upgraded from a launch note into a real multi-version public history.

Why it matters

This made it easier for a new visitor to understand:

  • Where to ask questions.
  • Where to share ideas.
  • Where to follow releases.
  • Where to read the official version history.
