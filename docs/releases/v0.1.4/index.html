Roadmap pass

v0.1.4 made the public story more future-aware.

Good open-source presentation is not just about launch day. Visitors also want to know what is current, what is next, and what is still out of scope.

Highlights

  • A roadmap page that says what is next without pretending the project is becoming a cloud sync product.
  • Stale announcement and release references cleaned up across the docs surface.
  • A stronger what-next path from the landing page into roadmap and release history.

Why it matters

v0.1.4 makes it easier for a new visitor to answer three questions quickly:

  • What is current?
  • What is likely next?
  • What is explicitly out of scope?
