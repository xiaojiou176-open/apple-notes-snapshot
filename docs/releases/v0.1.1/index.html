Discovery hardening

v0.1.1 made the launch surface easier to discover and share.

After the first public launch, this pass focused on metadata, structured data, and stronger public entry points.

Highlights

  • Richer GitHub Pages metadata with canonical, Open Graph, Twitter, favicon, manifest, sitemap, and robots support.
  • FAQ rich-result schema plus breadcrumb structured data across the public docs.
  • A public security.txt endpoint for scanners and automated security tooling.
  • Stronger community entry points in the README and public landing pages.
  • Refreshed release notes that point to live docs and discussions instead of placeholder paths.

Why it matters

v0.1.0 established the first public launch surface. v0.1.1 made that surface easier to discover, easier to share, and easier to navigate once people arrived.
