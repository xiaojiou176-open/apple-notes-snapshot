# Apple Notes Snapshot v0.1.12

## What changed after v0.1.11

This release turns the builder-side closeout work into tagged truth.

## Highlights

- promoted Codex and OpenClaw to a tagged v0.1.12 attach-proof trail
- added a standalone public skill packet for `notes-snapshot-control-room`
- added a real MCP Registry-ready lane with `server.json`, an MCPB manifest, a build
  script, and a locally verified `.mcpb` artifact path

## Why it matters

This release does not widen product scope. It makes the builder side feel more
finished. In plain language: the repo now has fresher named-host proof, a
portable public skill packet, and a real packaging lane for the official MCP
Registry without pretending every machine or registry lane is already done.
