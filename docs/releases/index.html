Current main branch

The latest public tag is currently v0.1.9. At the current baseline, main and the latest public tag point at the same hard-cut release commit. Future commits may move ahead again, so keep treating this section as a main-branch preview rather than a permanent promise that main always equals the latest public release.

The current tagged truth includes the clean public-repo hard cut, the control-room-first README and docs surface, the Local Web API and MCP pages, the dedicated builder packs, the public-skills pack, and the governance/closeout runbooks that keep release, proof, and maintainer language honest.
