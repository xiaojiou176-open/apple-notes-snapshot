# Apple Notes Snapshot v0.1.1

## What changed after the initial public launch

This release tightens the public discovery surface and the post-launch community
entry points around Apple Notes Snapshot.

## Highlights

- richer GitHub Pages metadata with canonical, Open Graph, Twitter, favicon,
  manifest, sitemap, and robots support
- FAQ rich-result schema plus breadcrumb structured data across the public docs
- a public `security.txt` endpoint for scanners and automated security tooling
- stronger community entry points in the README and public landing pages
- refreshed release notes that point to live docs and discussions instead of
  placeholder paths

## Why it matters

`v0.1.0` established the first public launch surface. `v0.1.1` makes that
surface easier to discover, easier to share, and easier to navigate once people
arrive.
