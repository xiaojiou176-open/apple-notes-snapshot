Latest release

v0.1.5 smooths out the first successful run.

This release improves the path from "I just found this repo" to "I got a believable local snapshot working on my Mac."

Highlights

  • A public examples page that shows believable local workflows.
  • A public troubleshooting page for permissions, launchd, stale-run, and Web UI issues.
  • Stronger navigation from the landing page, roadmap, support, and community pages into those surfaces.

Why it matters

People do not only need to understand a project. They also need to picture where it fits and what to do when the first run is not smooth. v0.1.5 reduces that gap.
